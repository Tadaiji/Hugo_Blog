How to install Unity#

How to install and setup Unity: Setting up Unity is an easy feat in today\’s time, because of the Unity-Hub. The Unity-Hub is a office Unity Software and allows you to install almost any Unity-Build with the necessary options to install all relevant development and deployment methods for the selected Unity-Build. So I went to the Unity Website.

From there I clicked on the “Get started” button and could sign up for the student or personal license. And if you are signed up or just logged in than you can download the Unity-Hub over your profile.

And after following the setup I got Unity-Hub, which looks something like this.

From here we need to install a Unity-Build. Now for the upcoming project I need an Android-build and a Windows build.
